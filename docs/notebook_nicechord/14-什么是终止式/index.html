14-什么是终止式

很多弹钢琴的人听到终止式这三个字就会联想到HANON的音阶练习的最后面的那四和和弦，甚至很多人以为那四个和弦就是所谓的终止式，但其实不是。

终止式，其实就是它字面上的意思，终止的方式。或者说是用来结束一句、一段、一首音乐的最后一两个和弦的安排方式。

终止式有很多很多种，这里介绍其中两中最常见的终止式。

正格终止
